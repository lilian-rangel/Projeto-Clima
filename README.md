<h1 align="center">Projeto Clima</h1> <br>

<p>Projeto para criação de uma aplicação de Clima com OpenWeather API.<br>
O objetivo é continuar aprofundando os conhecimentos em HTML5, CSS3, JavaScript e consumo de APIs.
</p>


<hr>
<h2 align="center">Aprendizados nesse projeto 🎯</h2>
Com esse projeto aprendi sobre como consumir dados da OpenWeather API e a utilizá-los de forma dinâmica.
